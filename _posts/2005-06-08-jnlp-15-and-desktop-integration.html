--- layout: post title: JNLP 1.5 and Desktop Integration published: true meta: {} tags: - Java type: post status: publish --- Java Webstart has a couple of very interesting [new features](http://weblogs.java.net/blog/stanleyh/archive/2005/06/java_deployment.html).Especially the improved desktop integration is very nice, featuring shortcuts creation, file extensions and MIME association and related content integration in the Start Menu.Additionally, on Windows Java Webstart Applications integrate in the Control Panel's Add/Remove Programs Panel.
